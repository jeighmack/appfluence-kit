# Appfluence Kit

The Digital Twin starter kit. Skills for a Hermes-based twin that briefs your morning, drafts your repetitive work, and keeps a pulse on your business.

**Install:** skills live in `~/.hermes/skills/`. Tell your twin: *"Install my Appfluence kit"* with this repo's URL, or copy the folders in manually.

Built by [Appfluence](https://appfluence.ai). Setup chain (`setup-my-twin` and friends) builds the twin; operating skills (`the-daily-briefing` is the heart) run it.
